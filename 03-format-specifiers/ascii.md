The ASCII (American Standard Code for Information Interchange) table is a character encoding standard for electronic communication. ASCII codes represent text in computers, telecommunications equipment, and other devices that use text. Each ASCII code corresponds to a character or a control function.

Here is a summary of the ASCII table, including the printable characters and some of the control characters:
